### ✨ Overview
Developed by team **Cache Cat**, this is an intelligent financial analysis tool designed to help stockholders make faster, smarter decisions. 📈 By automating the extraction of data from balance sheets, our app allows users to compare different stocks side-by-side with zero manual data entry. ⚡️

## ✨ Key Features
* **📄 Smart Parsing:** Upload raw balance sheets and let our app extract the vital numbers.
* **⚖️ Instant Comparison:** Compare two or more stocks across key metrics like P/E ratio, Debt-to-Equity, and Net Profit.
* **📊 Visualization:** Beautifully rendered charts to see financial trends over time.
* **💡 Decision Support:** Simplifies complex financial data into easy-to-read insights for everyday investors.
